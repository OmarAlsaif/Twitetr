Twitetr

Här har vi samlat all viktig information om hur man använder Twitetr för att skriva ett tweet, få det rättstavat och sedan skicka det på Twitter.

API- Beskrivning

Vårt API tar emot en sträng från parametern "userTweet" och skickar vidare detta till Libris API. Det som tas emot från Libris API är en XML fil som visas i exmplet nedan. Returdatan tas emot av ett program kallat "Velocity Template Engine" som sparar datan som skickas av servern i som en array av strängar

GET /

Visar ett användargränssnitt för att mata in ett tweet.

Indata

Ej möjligt för GET

GET /tweet{userTweet}

Skickar användarens tweet som en sträng till Libris API och hämtar rättstavade versionen av strängen i en XML fil, som visas i exmplet nedan.

Indata

En sträng från användaren.
